# Halcyon Theme

A minimal, dark blue theme for VS Code, Sublime Text, Atom, iTerm, and more.

![Halcyon Theme Screenshot](https://via.placeholder.com/800x400)

## Features

- **Minimalist Design**: Distraction-free coding environment.
- **Eye-friendly**: Dark blue palette reduces eye strain.
- **Wide Support**: Available for multiple editors and terminals.

## Installation

### VS Code

1. Open **Extensions** sidebar panel in VS Code. `View â†’ Extensions`
2. Search for `Halcyon Theme`
3. Click **Install** to install it.
4. Code > Preferences > Color Theme > **Halcyon Theme**

### Sublime Text

1. Install `Halcyon Theme` via Package Control.
2. Add the following to your user settings:
    ```json
    "theme": "Halcyon.sublime-theme",
    "color_scheme": "Packages/Halcyon Theme/Halcyon.tmTheme"
    ```
